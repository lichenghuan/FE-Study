### :question: 如何学习
- 不管懂与不懂都要先思考再百度，思考后一定要去写出来
- 如果是原生js的题，不要依赖使用第三方库，如vue等
- 每天的题目都是独立的，不需要按顺序来，但建议每道题都去答下，有你会的和有你不会的，答了才知道
- 不懂的题百度学习后，不要直接复制一大段过来，要用自己的话精简地总结概括出来
